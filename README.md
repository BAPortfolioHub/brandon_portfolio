# Portfolio Website

## Description

This is a personal portfolio website built using React, TypeScript, and CSS. The website showcases my projects, skills, and experience in a professional and visually appealing manner.

## Installation

To get the project up and running locally, follow these steps:

1. **Fork the repository:**

2. **Clone your forked version:**
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   cd portfolio-website
   ```
3. **Install Dependencies**:
    Ensure you have node installed locally and run:
    ```bash
    npm install
    ```
4. **Run the project locally**:
    ```bash
    npm start
    ```
5. **Open your browser**:
    Open your browser you `http://localhost/3000` to view the page

## Usage
The website will automatically reload if you make changes to the code.
- You can edit the content of the website by modifying the appropriate React components in the src directory.
- Be sure to upload your appropriate profile photo into the `src/assets` folder.

- To build the project for production, run:
```bash
npm run build
```
## Features
- Responsive Design: The website is fully responsive and looks great on all devices.
- Dynamic Content: Projects and skills can be easily updated through the code.
- Modern Stack: Built with React and TypeScript for a robust and maintainable codebase.
- Styling: Custom CSS is used for styling the components.
